# 股票数据解析工具

一个基于 React + Vite + TailwindCSS 开发的股票数据解析 Web 应用，具有美观的科技感 UI 和良好的交互体验。

## 功能特性

### 页面1: HTML表格解析
- 输入 HTML 代码，自动解析其中的表格数据
- 支持解析多个表格
- 完整提取所有表格数据，不遗漏任何信息
- 一键复制表格数据（制表符分隔格式）

### 页面2: 股票信息查询
- 输入股票代码或名称查询股票信息
- 显示股票价格、涨跌幅、成交量等关键信息
- 展示股票所属行业和概念板块

## 技术栈

- **React 18** - UI 框架
- **Vite** - 构建工具
- **TailwindCSS** - CSS 框架
- **React Router** - 路由管理

## 使用说明

### HTML表格解析

1. 在左侧输入框中粘贴包含表格的 HTML 代码
2. 点击"解析表格"按钮
3. 右侧将显示解析后的表格数据
4. 点击"一键复制"按钮可复制所有表格数据

### 股票查询

1. 在输入框中输入股票代码（如：000001）或股票名称（如：平安银行）
2. 点击"查询"按钮或按 Enter 键
3. 查看显示的股票详细信息

**注意**: 当前股票查询功能为演示模式，仅支持部分股票代码。实际应用中需要接入真实的股票数据 API。
